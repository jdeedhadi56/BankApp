# 🏦 BankApp - تطبيق إدارة البنك
## 🖥️ واجهة التطبيق
التطبيق يعمل بالكامل عبر **الشاشة السوداء (Console Application)** ولا يحتوي على واجهة رسومية (GUI). يمكن تشغيله على الكمبيوتر عبر الـ Command Prompt / Terminal، أو على الهاتف باستخدام تطبيق **Termux**.

تطبيق بنكي مصمم بلغة ++C لإدارة الحسابات والعملاء من خلال ملفات نصية.

## 🛠️ شروط ومتطلبات التشغيل الدقيقة

لكي يعمل التطبيق بشكل صحيح وبدون أخطاء، **يجب** اتباع الشروط التالية قبل تشغيل الملف التنفيذي:

1-**الملفات التلقائية:** 
   - يقوم التطبيق تلقائيًا بإنشاء ملف للمستخدمين وملف للعملاء داخل مجلد `BankApp`.

## 🔐 نظام تسجيل الدخول الافتراضي (صلاحيات المسؤول)

للدخول إلى النظام لأول مرة كمسؤول (Admin)، يجب كتابة السطر التالي داخل ملف المستخدمين:
```text
admin#//#1234#//#-1
```

### 💡 قواعد بيانات تسجيل الدخول:
* **اسم المستخدم الافتراضي:** `admin`
* **كلمة المرور الافتراضية:** `1234`
  * **الصلاحيات : '1-' تعني صلاحيات كاملة
* **ملاحظة هامة:** يمكنك تغيير الاسم وكلمة المرور إلى أي اسم أو رقم آخر تريد، بشرط وضع الفاصل الزمني **`#//#`** بينهما (مثال: `User#//#MyPassword99`).

## 🚀 كيفية التشغيل
1. تأكد من إنشاء مجلد `BankApp`.
2. قم بتشغيل الملف التنفيذي للتطبيق.
3. أدخل بيانات الأدمن الموضحة أعلاه للولوج إلى اللوحة الرئيسية.
